# SyncCanvas DBClient

## 项目概括
SyncCanvas DBClient 是一个基于 C++ 和 Qt 6.5 开发的桌面客户端应用程序。
目前项目处于早期开发阶段，主要包含主窗口框架、动态气泡背景特效以及欢迎界面组件。

## 技术选型
- **编程语言**: C++ 17+
- **GUI 框架**: Qt 6.5 (Widgets 模块)
- **构建系统**: CMake 3.19+
- **资源管理**: Qt Resource System (qrc)

## 项目结构 / 模块划分
- `main.cpp`: 应用程序入口
- `mainwindow.*`: 主窗口逻辑，包含气泡背景动画实现
- `welcomewidget.*`: 欢迎界面组件 (无边框窗口，透明背景)
- `global.*`: 全局变量与定义
- `res.qrc` & `res/`: 应用程序资源文件 (图标、图片等)

## 核心功能 / 模块详解
- **主界面 (MainWindow)**: 
  - 承载应用主要布局。
  - 实现自定义 `paintEvent` 绘制动态气泡背景。
  - 管理子窗口切换。
- **欢迎页 (WelcomeWidget)**:
  - 启动时的引导或登录界面。
- **资源系统**: 
  - 集成应用所需的静态资源 (如 `offline.png`, `shared.png`)。

## 技术实现细节
- **动态背景**: 使用 `QPainter` 在 `paintEvent` 中绘制，结合 `QTimer` 定时更新气泡坐标实现动画效果。
- **资源修复**: 强制开启 `CMAKE_AUTORCC` 解决资源文件未正确打包的问题。
